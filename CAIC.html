Grand County 2020 Forest Fire Remote Analysis

This analysis was done completely in ARCgisPRO. The intent of the project was to complete a remote sensing analysis of the wildfire near Grand County in 2020.

This map was made using Google Earth Engine Tree Cover / Loss and NDVI from Landsat. One of the maps uses USGS EE Lvl 3 Science Product "Burned Area"
